# Product Studio

**Decision-Support for Product Managers**

Product Studio is a structured thinking framework tool designed to help Product Managers reason through ambiguous product situations. It provides frameworks for critical thinking, but does NOT automate decisions or provide recommendations.

## üéØ Purpose

Help Product Managers make better decisions by:
- Structuring complex problems
- Analyzing root causes
- Evaluating risks systematically
- Comparing tradeoffs
- Planning next steps responsibly

**Important:** This tool supports your judgment; it does NOT replace it.

## ‚ú® Core Capabilities

### 1. Problem Framing
Frame your problem clearly to ensure you're solving the right issue.
- Define the problem statement
- Identify affected parties
- Understand the impact
- Clarify timing and urgency
- Surface key assumptions

### 2. Root Cause Analysis
Dig deeper to understand the underlying causes using:
- 5 Whys methodology
- Contributing factors analysis
- Pattern identification

### 3. Risk Management
Systematically identify and evaluate risks:
- Technical risks
- Business risks
- User experience risks
- Organizational risks
- Mitigation strategies

### 4. Tradeoff Evaluation
Compare options and understand their tradeoffs:
- Side-by-side option comparison
- Multi-criteria evaluation
- Cost-benefit analysis
- Decision factors weighting

### 5. Responsible Next Steps Guidance
Plan next steps with appropriate caution:
- Confidence assessment
- Information gap identification
- Validation opportunities
- Stakeholder alignment

## üöÄ Getting Started

### Running Locally

1. Clone this repository:
   ```bash
   git clone https://github.com/anubhav-product/ProductPlayGround.git
   cd ProductPlayGround
   ```

2. Open `index.html` in your web browser:
   ```bash
   # On macOS
   open index.html
   
   # On Linux
   xdg-open index.html
   
   # On Windows
   start index.html
   ```

3. Or use a local server:
   ```bash
   # Python 3
   python -m http.server 8000
   
   # Python 2
   python -m SimpleHTTPServer 8000
   
   # Node.js (if you have http-server installed)
   npx http-server
   ```

4. Navigate to `http://localhost:8000` in your browser
